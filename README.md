# Kotlin CLI Todo App

A command-line Todo application built with Kotlin using Object-Oriented Programming (OOP) concepts.

## Features

- Create and manage todo items.
- Store todo data locally using `HashMap`.
- Store notes in MongoDB.
- Retrieve notes from MongoDB.
- Modify existing notes in MongoDB.

## OOP Concepts Used

- Private variables for encapsulation.
- Private classes accessed through interfaces.
- Interface-based abstraction.

## Data Storage

### Local Storage

- Uses `HashMap` to store todo data locally.

### MongoDB

- Store notes.
- Retrieve notes.
- Modify existing notes.
